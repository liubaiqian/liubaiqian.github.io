简易基岩版MC服务器配置文件编辑器


By FH3718，制作于2025.7.7，最后更新于2025.7.7
此项目旨在让新服主可以零基础编辑自己服务端的配置文件(server.properties)


服务器名称
游戏模式（ survival生存，creative创造，adventure冒险）
难度（"peaceful"和平, "easy"简单, "normal"普通, "hard"困难）
允许作弊
最大玩家数
在线模式(建议开启)
白名单模式(如果你不知道这个选项的作用请勿勾选)
服务器端口（IPv4）(默认为19132)
服务器端口（IPv6）(默认为19133)
渲染距离(依据服务器性能填写 建议4-64)
模拟距离(建议4-16)
挂机踢出时间(关闭该功能请填写0)
服务端使用的最大线程数(填写0为无限制)
存档名称
世界种子(只对新创建的世界生效，空留为随机)
默认玩家权限("visitor"访客, "member"成员, "operator"管理员)
强制同步材质包(不知道这个选项的作用请勿勾选)
